# 文件仓库

一个简单的网页应用，用来上传文件并自动生成 jsdelivr CDN 加速链接。

## 功能

- 📤 上传文件（支持任何格式和大小）
- 🔗 自动生成 jsDelivr CDN 链接
- 💾 本地保存上传记录
- 🚀 CDN 加速，全球快速访问

## 使用

1. 打开 `index.html`
2. 选择或拖拽文件
3. 自动生成链接，复制即用

## 文件结构

- `index.html` - 主页面
- `style.css` - 样式表
- `app.js` - 应用逻辑
- `uploads/` - 上传的文件目录
